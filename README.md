# Maho_Ai

[![CI](https://github.com/mageaustralia/maho-module-ai/actions/workflows/ci.yml/badge.svg)](https://github.com/mageaustralia/maho-module-ai/actions/workflows/ci.yml)
[![License: OSL-3.0](https://img.shields.io/badge/license-OSL--3.0-blue.svg)](LICENSE)

> Core Maho AI platform — multi-provider LLM access with an async queue, safety guardrails, and admin tooling.

## Status

🚧 **Pre-release / WIP** — installable via composer but not yet tagged. API and behaviour may change before v1.0.

## Requirements

- Maho 26.3+
- PHP 8.3+

## Installation

```bash
composer require mageaustralia/maho-module-ai:dev-main
composer dump-autoload -o
```
